KEY FEATURES:

✅ 4-Panel Layout
   - Map (14 lines, green) - Shows star positions with fog-of-war
   - Game State (15 lines, blue) - Shows controlled stars & fleets tables
   - Reports (flexible, cyan) - Scrollable history of events & feedback
   - Input (6 lines, yellow) - Command entry with inline status

✅ Scrollable Panels
   - Game State: Scrolls if tables exceed 15 lines
   - Reports: Always scrollable, maintains full history
   - Auto-scrolls to latest message

✅ Rich Feedback
   - Success messages in GREEN
   - Error messages in RED
   - Info messages in default color
   - Combat reports with emojis (⚔️)
   - Hyperspace losses with emojis (⚡)

✅ Input Integration
   - Real-time validation
   - Commitment tracking
   - Order queueing with count
   - Clear error messages

✅ Game State Updates
   - Update all panels atomically
   - Combat reports auto-display
   - Hyperspace losses auto-display
   - Map refreshes with new visibility

COMMANDS:
  move <ships> from <star> to <star>  - Queue an order
  done                                 - Submit orders & end turn
  list                                 - Show queued orders
  clear                                - Clear order queue
  help                                 - Display help
  status                               - Refresh game state
  quit                                 - Exit game

EXAMPLE SESSION:
  > help                      → Shows help in reports
  > move 5 from A to B        → ✓ Order 1 queued: 5 ships from A to B
  > move 100 from A to C      → ❌ Error: Insufficient ships at A
  > list                      → Shows queued orders in reports
  > done                      → Submitted 1 order
  > quit                      → Exits TUI
